# Generic Tree - Português

Estrutura de dados de árvore n-ária flexível para Python.

## Início Rápido

```python
from generic_tree import GenericTree

tree = GenericTree("raiz")
tree.add_child("filho1")
tree.add_child("filho2")
print(tree)
```

## Características

- Estrutura n-ária completa
- Suporte para qualquer tipo de valor Python
- Suporte a metadados
- Múltiplas operações de travessia
- Busca e filtragem eficientes
- Serialização
