# Tabela Multidimensional - Português

Operações eficientes em arrays/tabelas multidimensionais.

## Início Rápido

```python
from multitable import MultiTable

table = MultiTable(2)  # Tabela 2D
table[0][0] = 42
print(table[0][0])  # Saída: 42
```

## Características

- Suporte a qualquer dimensão
- Operações de transposição
- Remodelagem e achatamento
- Operações estatísticas
- Fusão e combinação
