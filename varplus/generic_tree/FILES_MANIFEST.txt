USAGE QUICK REFERENCE
=====================

Basic Creation:
    tree = Tree(root_value="root")
    child = tree.add_child(tree.root, "child")

Traversal:
    for node in tree.traverse(TraversalMode.PRE_ORDER):
        print(node.value)

Search:
    node = tree.get_node_by_value("target")
    nodes = tree.get_nodes_by_predicate(lambda x: x > 5)

Serialization:
    tree.save_to_file("tree.json")
    tree = Tree.load_from_file("tree.json")

Functional:
    new_tree = tree.map(lambda x: x * 2)
    filtered = tree.filter(lambda x: x > 0)
    total = tree.reduce(lambda acc, val: acc + val)

Analysis:
