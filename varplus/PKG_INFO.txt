# Generic Tree

A complete, production-ready n-ary tree data structure implementation for Python.

## Features

- Support for all Python data types as node values
- N-ary tree structure (unlimited children per node)
- Multiple traversal modes (PRE_ORDER, POST_ORDER, LEVEL_ORDER, IN_ORDER)
- Complete search algorithms (DFS, BFS, value, predicate)
- Functional operations (map, filter, reduce, apply)
- Tree analysis (height, depth, balance, leaves)
- JSON serialization
- Zero external dependencies
- Type hints (100%)
- Complete docstrings (100%)
- 337+ unit tests (100% pass rate)

## Quick Start

```python
from generic_tree import Tree, TraversalMode

# Create a tree
tree = Tree(root_value="root")

# Add children
child1 = tree.add_child(tree.root, "child1")
child2 = tree.add_child(tree.root, "child2")

# Traverse the tree
for node in tree.traverse(TraversalMode.PRE_ORDER):
    print(node.value)
```
