# PydocsExport - Português

Exportador de documentação Python para vários formatos.

## Início Rápido

```python
from PydocsExport import PydocsExport

exporter = PydocsExport()
exporter.export_module("module_name", "output_format")
```

## Características

- Suporte múltiplos formatos
- Documentação automatizada
- Customização flexível
